# 花朵与花园规划 v1

这份文档用于后续开发素材准备：先定义 5 个花园主题，每个花园 12 朵花，其中 9 朵普通花、3 朵高级花。分类逻辑优先按自然属性展开，包括地域、季节、气候带、湿度与生境，让每个花园的视觉气质和素材差异足够明确。

## 规划思路

### 1. 花园先按自然属性分层

5 个花园覆盖不同自然环境：

| gardenId | 花园名称 | 地域/生境 | 气候属性 | 主季节 | 视觉关键词 |
| --- | --- | --- | --- | --- | --- |
| spring_meadow | 北境春原 | 北温带草甸、林缘 | 温带/冷温带 | 春季 | 柔绿、嫩黄、清晨露水、小花簇 |
| mist_jiangnan | 江南雨庭 | 东亚江南水岸、庭院 | 亚热带湿润 | 春末至夏季 | 雨雾、粉白、青瓦、水汽 |
| mediterranean_sun | 地中海夏园 | 地中海海岸、石坡 | 夏干型亚热带 | 夏季 | 暖阳、银叶、蓝紫、陶土色 |
| tropical_cloud | 热带云岛 | 热带/亚热带雨林、云雾林 | 热带湿润 | 雨季/全年 | 高饱和、异形花、阔叶、荧光感 |
| alpine_starlight | 高山星谷 | 高山草甸、雪线附近 | 高海拔冷凉 | 短夏至初秋 | 矮生、星形、冷色、清透空气 |

### 2. 普通花与高级花的区分

普通花用于主线收集，轮廓清楚、形态亲切、适合大量制作 4 阶段成长图。高级花用于付费、成就、稀有掉落或活动奖励，必须具备更明显的外观特征：

- 普通花：花型易读，颜色不超过 2-3 个主色，适合 1024 x 1024 PNG 批量生产。
- 高级花：轮廓、色彩、结构、光效或动态潜力更强，缩略图下也能一眼认出。
- 高级花建议增加 idle 动效，例如微光、花瓣展开、露珠闪烁、藤蔓轻摆。

### 3. 素材命名建议

静态花朵仍沿用现有命名方式：

```text
{gardenId}_{flowerSlug}_{stage}.png
```

阶段固定：

```text
seed
sprout
bud
bloom
```

示例：

```text
spring_meadow_snowdrop_bloom.png
tropical_cloud_bird_of_paradise_bud.png
```

## 开发接入建议

### 数据结构建议

后续如果要替换或扩展 `Garden.catalog`，建议每朵花不要只存中文名，而是增加资源字段：

```swift
struct FlowerDefinition: Identifiable, Codable, Equatable {
    let id: String
    let name: String
    let slug: String
    let rarity: FlowerRarity
    let regionTag: String
    let seasonTag: String
    let climateTag: String
}
```

稀有度：

```swift
enum FlowerRarity: String, Codable {
    case ordinary
    case premium
}
```

### 解锁节奏建议

- 前 3 个花园可以作为主线免费成长路径：北境春原、江南雨庭、地中海夏园。
- 热带云岛适合作为视觉冲击更强的中后期花园，可加入活动或高级版奖励。
- 高山星谷适合作为后期成就花园，用更冷、更稀有的花强化完成感。
- 每个花园 9 朵普通花可作为完整收集目标，3 朵高级花用于高级版、活动、连续步数奖励或特别任务。

### 美术生产优先级

第一批建议先做：

1. 每个花园 3 朵普通花，用来测试花园差异和缩略图识别。
2. 每个花园 1 朵高级花，用来测试高级花的复杂度与动效预算。
3. 每个花园 1 张花园岛或背景主视觉，用来确认整体色彩系统。

第一批推荐样本：

| 花园 | 普通花样本 | 高级花样本 |
| --- | --- | --- |
| 北境春原 | 雪滴花、番红花、勿忘我 | 黑郁金香 |
| 江南雨庭 | 紫藤、荷花、绣球 | 牡丹 |
| 地中海夏园 | 薰衣草、金盏菊、向日葵 | 火焰百合 |
| 热带云岛 | 鸡蛋花、红姜花、大花美人蕉 | 天堂鸟花 |
| 高山星谷 | 龙胆、雪绒花、高山石竹 | 雪莲 |

## 风格控制

- 不要让 5 个花园只靠颜色区分，每个花园都要有不同的花型结构：草甸小花、水岸大叶、香草花穗、热带异形、高山垫状。
- 普通花尽量保持 1 个主要识别点，高级花至少有 2-3 个识别点，例如大轮廓、特殊纹理、罕见颜色、发光或动态结构。
- 同一花园内颜色要有主色和点缀色，避免 12 朵花全部同色。
- 花朵 bloom 阶段要能在小尺寸图鉴里读出来，bud 阶段要明显不同于 bloom，避免只是缩小版成花。
